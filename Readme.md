## Overview
This Python program performs two main tasks:

1. **File Reading:**  
   Reads data files in the following formats:
   - CSV (`.csv`)
   - Excel (`.xls`, `.xlsx`)
   - Parquet (`.parquet`)  

2. **Source-Target Row Count Check:**  
   Compares the number of rows between a **source** file and a **target** file and reports whether they match.  
   Useful for **data validation** or **ETL testing**.
